# Enyalien (에니알리엔)

Enyalien(엘프어로 '기억' 혹은 '회상'을 뜻함)은 FastAPI, SQLite (via SQLModel), HTMX, 그리고 Tailwind CSS (with DaisyUI)를 기반으로 구축된 Anki 스타일의 Spaced Repetition System (SRS) 개인용 암기카드 웹앱입니다.

에빙하우스 망각곡선 모델과 SM-2 알고리즘을 결합하여, 사용자가 망각하기 직전 타이밍에 카드를 복습할 수 있도록 스케줄링해 줍니다. 복잡한 SPA 프레임워크를 사용하지 않고 HTMX와 서버사이드 렌더링(SSR)을 결합하여 가볍고 빠른 반응 속도를 보장합니다.

또한, 아미노산 20종에 대한 3문자 및 1문자 약자를 외울 수 있는 학습 데이터가 기본으로 포함되어 제공됩니다.

---

## 🌟 주요 기능 (Features)

1. 일괄 카드 입력 및 마크다운 지원 (Bulk Import):
   - 카드를 하나씩 등록하는 번거로움 없이, 텍스트 입력 영역에 `[앞면]:[뒷면]` 형태로 여러 줄에 걸쳐 작성하면 여러 장의 카드를 일괄 등록해 줍니다.
   - 앞면과 뒷면 내용에는 마크다운 포맷이 완벽히 지원됩니다.

2. 역방향 복습 모드 (Reverse Review Mode):
   - 복습 페이지 상단의 셀렉트 박스에서 기본 복습 (앞면 ➡️ 뒷면) 혹은 역방향 복습 (뒷면 ➡️ 앞면) 모드를 변경할 수 있습니다.
   - 역방향 복습 모드에서는 카드의 뒷면(정답/설명)이 질문으로 표시되고, 앞면(단어/주제)을 맞추는 형태로 학습 방향을 전환하여 복습 강도를 조절할 수 있습니다.

3. 반응형 듀얼 테마 (라이트 & 다크):
   - 가독성이 높고 모던한 Dark (다크) 테마와 Light (라이트) 테마를 지원합니다.
   - 기본 테마는 Dark (다크) 테마로 설정되어 있으며, 상단 헤더의 태양/달 아이콘 버튼으로 자유롭게 테마를 전환할 수 있습니다. 사용자의 선택은 LocalStorage에 저장되어 유지됩니다.

4. 강력한 복습 상호작용 (단축키 & 제스처):
   - 데스크톱:
     - `Space` / `Enter`: 카드 뒤집기 (정답 확인)
     - `왼쪽 화살표 (←)`: 모름 (Again) 입력
     - `오른쪽 화살표 (→)` 또는 뒤집힌 상태에서 `Space` / `Enter`: 알고 있음 (Good) 입력
   - 모바일:
     - 카드를 탭하여 뒤집은 뒤, 왼쪽으로 스와이프하면 모름 (Again), 오른쪽으로 스와이프하면 알고 있음 (Good)으로 평가 및 제출됩니다.
     - 드래그 중인 방향에 따라 카드 테두리에 녹색(알고 있음)/적색(모름)의 글로우 효과와 부드러운 3D 회전 애니메이션이 제공됩니다.

5. 망각곡선 기반 스케줄링:
   - 사용자의 결과 제출("모름" / "알고 있음")에 따라 다음 복습 주기와 카드의 난이도 가중치(Ease Factor)를 계산하여 최적의 복습 시점을 산출합니다.

## 📈 SM-2 알고리즘 규칙

복습 시점 계산은 SM-2 알고리즘을 단순화하여 "모름"과 "알고 있음" 두 가지 선택지에 매핑했습니다.

- 모름 (Again):
  - 연속 성공 횟수를 `0`으로 리셋합니다.
  - 다음 복습 주기를 즉시 다음 날(`1일`)로 세팅합니다.
  - 카드의 쉬움 정도(Easiness Factor)를 `0.2`만큼 차감하여 더 자주 등장하게 만듭니다. (최소값 1.3)
- 알고 있음 (Good):
  - 연속 성공 횟수를 `1` 증가시킵니다.
  - 다음 복습 주기(Interval) 계산:
    - 1회차 성공: `1일` 뒤
    - 2회차 성공: `6일` 뒤
    - 3회차 이상: `이전 주기 * Easiness Factor` 만큼 날짜 증가
  - Easiness Factor를 `0.15`만큼 누적 증가시켜 복습 주기가 더 빠르게 멀어지도록 만듭니다. (최대값 3.0)

모든 다음 복습 예정 시점(`next_review`)은 정각 기준 시간대로 보정되어 분 단위 스케줄링 꼬임을 방지합니다.

## 🚀 실행 및 배포 가이드

### 1. 로컬 환경에서 구동하기

의존성 설치 및 실행 (uv 사용):
```bash
# 가상 환경 생성 및 동기화
uv sync

# 서버 실행
uv run python main.py
```

실행 후 브라우저에서 `http://localhost:8000`에 접속합니다.

#### 기본 아미노산 20종 데이터 추가

데이터베이스에 아미노산 20종 약자 데이터를 추가 하려면 아래 스크립트를 한 번 실행해 주세요:

```bash
uv run python scripts/seed_amino_acids.py
```

#### 데이터베이스 초기화 (Reset)

등록된 모든 암기카드 및 복습 이력을 완전히 삭제하고 깨끗한 데이터베이스 상태로 리셋하려면 아래 스크립트를 실행해 주세요:
```bash
uv run python scripts/reset_db.py
```

#### 데이터베이스 백업 (Backup)

현재 저장된 모든 암기카드를 마크다운(Markdown) 포맷 파일로 안전하게 내보내려면 아래 스크립트를 실행해 주세요:
```bash
uv run python scripts/backup_db.py
```
생성된 백업 파일(`enyalien_backup_*.md`) 내부의 Raw Bulk Data를 복사하여 나중에 다른 기기에서 일괄 카드 추가(Bulk Import) 용도로 그대로 재사용할 수 있습니다.

## 🐳 Docker 배포 (개인 웹서버 배포용)

Docker 컨테이너가 교체되거나 업데이트되어도 데이터베이스가 유실되지 않도록 볼륨 마운트를 함께 세팅해야 합니다.

`docker-compose.yml`이 이미 구성되어 있으므로 아래 명령어로 간단히 백그라운드 배포를 수행할 수 있습니다.
```bash
# Docker 이미지 빌드 및 서비스 백그라운드 가동
docker compose up -d --build
```
- 호스트의 `8000` 포트로 바인딩됩니다. (`http://서버IP:8000`)
- SQLite 데이터베이스는 호스트 머신의 네임드 볼륨 `enyalien_data`에 매핑되어 컨테이너의 `/data/enyalien.db` 경로에 안전하게 영구 보관됩니다.
